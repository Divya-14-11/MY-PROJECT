* **Classification**

    5. 2. **Unsupervised Learning**

    * **Definition**:
    The algorithm learns patterns from **unlabeled data**, identifying structure or relationships.

    * **Goal**:
    Discover hidden patterns or data groupings.

    * **Common Tasks**:
    1. * **Clustering** (e.g., customer segmentation)
    2. * **Dimensionality Reduction** (e.g., PCA, t-SNE)

    * **Examples**:
    1. * K-means clustering
    2. * Hierarchical clustering
    3. * Principal Component Analysis (PCA)
    4. * Autoencoders

    ### 3. **Reinforcement Learning**

    ***Definition**:
    The algorithm learns by interacting with an environment and receiving feedback in the form of **rewards** or **penalties**.

    * **Goal**:
    Learn a policy to maximize cumulative reward over

    * **Common Applications**:
    1. * Game playing (e.g., AlphaGo)
    2. * Robotics
    3. * Autonomous driving

    * **Examples**:
    1. * Q-learning
    2. * Deep Q-Networks (DQN)
    3. * Policy Gradient methods

    ### 4. **Self-Supervised Learning** (Emerging Category)

    * **Definition**:
    A form of supervised learning where labels are **generated automatically**from the input data itself.

    * **Goal**:
    Learn useful representations without requiring labeled data. * **Used in**:
    1. * Natural Language Processing (e.g., BERT, GPT)
    2. * Computer Vision (e.g., contrastive learning)

    * **Examples**:
    1. * Contrastive learning (SimCLR, MoCo)
    2. * Masked language modeling (BERT)
